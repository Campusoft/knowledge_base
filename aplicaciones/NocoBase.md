# NocoBase 

NocoBase is a scalability-first, open-source no-code/low-code platform for building business applications and enterprise solutions.


Frontend 
- React
- Ant Design	
- Formily
- Axios
- i18next
- react-router-dom	

Backend 
- Koa.js	
- Sequelize
- Umzug
- i18next
- ws WebSocket support

Development Tools

- Lerna NocoBase uses a monorepo architecture managed by Lerna and Yarn Workspaces.
