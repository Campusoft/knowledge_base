# Bruno

Bruno is a Git-friendly and offline-first open-source API client aimed at revolutionizing the status quo represented by tools like Postman and Insomnia.

Bruno es un cliente API de código abierto, gratuito y ligero, diseñado para explorar, probar y gestionar APIs de forma eficiente. Es una alternativa moderna a herramientas como Postman e Insomnia, con un enfoque particular en la privacidad, el control de versiones y el trabajo offline


Características Principales

- Código abierto y gratuito: Bruno es completamente open source y la mayoría de sus funciones son gratuitas, con versiones comerciales opcionales para equipos que necesitan características adicionales.

- 100% offline: No requiere conexión a internet ni cuenta de usuario. No hay sincronización en la nube, lo que garantiza que tus datos (incluyendo tokens, claves y respuestas de API) permanezcan siempre en tu dispositivo local.

- Integración nativa con Git: Las colecciones de API se guardan directamente en carpetas del sistema de archivos como archivos de texto plano en formato .bru. Esto permite usar Git o cualquier sistema de control de versiones para colaborar, rastrear cambios y mantener sincronizados los equipos de desarrollo.

- Rápido y ligero: Consume muy pocos recursos y carga en fracciones de segundo, a diferencia de clientes API más pesados.
Multiplataforma: Funciona en Windows, macOS y Linux
